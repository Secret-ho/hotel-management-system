Hotel Booking Website
A university project that demonstrates the design of a hotel booking website interface along with a database schema for managing hotel reservations. The project focuses on frontend development and database structure for handling booking-related data.

Project Overview
This project simulates a hotel reservation platform where users can browse hotel information and interact with booking-related interfaces. The repository also includes a database design that models how reservation data could be stored and managed.

The goal of this project is to practice:

Frontend web development
UI design for booking systems
Database schema design for reservation management
Technologies Used
HTML
CSS
JavaScript
SQL (Database Schema)
