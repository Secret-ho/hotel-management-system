# Hotel Booking System

A university project that demonstrates the design and implementation of a **Hotel Booking Website Interface** along with a structured **Database Schema** for managing reservations.

> This project focuses on **frontend development** and **relational database design** for a hotel reservation system.

---

## Project Overview

The system simulates a hotel booking platform where users can:

- View hotel and room information
- Access booking interfaces
- Interact with a reservation form

The repository also includes a well-structured SQL database schema to manage:

- Hotels
- Rooms
- Customers
- Reservations

## Database Design

The database schema includes the following core entities:

- **Hotels**
- **Rooms**
- **Customers**
- **Reservations**

The design demonstrates relationships such as:

- One hotel → Multiple rooms
- One customer → Multiple reservations
- Each reservation → Linked to a specific room
